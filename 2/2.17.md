# 2.17

last-pair

途中で作成したlist-ref、lengthを利用した

```scheme
(define (list-ref items n)
    (if (= n 0)
        (car items)
        (list-ref (cdr items) (- n 1))))

(define (length items)
    (if (null? items)
        0
        (+ 1 (length (cdr items)))))

(define (last-pair items)
    (list (list-ref items (- (length items) 1))))
```
