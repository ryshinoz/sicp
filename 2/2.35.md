# 2.35

count-leaves 

```scheme
(define (count-leaves x)
    (cond ((null? x) 0)
          ((not (pair? x)) 1)
          (else (+ (count-leaves (car x))
                   (count-leaves (cdr x))))))
```

```scheme
(define (count-leaves t)
  (accumulate (lambda (x y) (+ y 1))
              0
              (map (lambda (x) (if (pair? x) 
                                   (count-leaves x)
                                   x)) t)))
```
